struct MatePropertyBox

struct MatePropertyBox;

Warning

MatePropertyBox is deprecated and should not be used in newly-written code.

An opaque widget representing a property box. Items should be added to this widget using mate_property_box_append_page.

Warning

This sets the modified flag of the MatePropertyBox to the value in state. Affects whether the OK/Apply buttons are sensitive.
